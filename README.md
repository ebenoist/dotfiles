Erik's Dotfiles
===================

Install
-------

Clone onto your laptop:

    git clone git://github.com/ebenoist/dotfiles.git

Install:

    cd dotfiles
    ./install.sh

This will create symlinks for config files in your home directory.

You can safely run `./install.sh` multiple times to update.
